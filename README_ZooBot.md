# Moscow Zoo Quiz Bot 🐾

Telegram-бот викторина для популяризации программы опеки Московского зоопарка.  
Проект помогает пользователям узнать своё «тотемное животное», познакомиться с особенностями животных зоопарка и узнать больше о программе опеки.

---

# 📌 О проекте

Бот разработан в формате интерактивной викторины для Telegram.  
Пользователь отвечает на серию вопросов, после чего получает персонализированный результат с описанием животного, стикером и информацией о программе опеки.

Проект ориентирован на:
- вовлечение аудитории Telegram-канала зоопарка;
- популяризацию программы опеки животных;
- создание интерактивного и развлекательного контента.

---

# ✨ Возможности бота

## 🎮 Интерактивная викторина
- пошаговое прохождение вопросов;
- несколько вариантов ответов;
- система подсчёта баллов;
- определение тотемного животного пользователя.

## 🐻 Персонализированный результат
После завершения викторины пользователь получает:
- описание своего животного;
- тематический стикер;
- информацию о программе опеки.

## 📤 Поддержка социальных сетей
Результатом можно поделиться:
- в Telegram;
- во ВКонтакте;
- в Одноклассниках.

Публикация содержит ссылку на бота для привлечения новых пользователей.

## 💬 Обратная связь
Пользователь может:
- оставить отзыв;
- связаться с сотрудником зоопарка;
- получить дополнительную информацию.

## 🔁 Повторное прохождение
После завершения викторины доступна кнопка:
- «Попробовать ещё раз».

## 🧹 Автоматическая очистка сообщений
Бот автоматически удаляет устаревшие сообщения:
- старые вопросы;
- справочные сообщения;
- результаты предыдущих викторин.

Это делает интерфейс более чистым и удобным.

# 🏗 Архитектура проекта

Проект построен по модульной структуре:

```text
handlers/      # обработчики команд и действий
services/      # бизнес-логика
data/          # вопросы, животные, стикеры
database/      # SQLite база данных
keyboards/     # inline/reply клавиатуры
utils/         # логирование и вспомогательные функции
```

# 🗂 Структура данных

## Викторина
Вопросы и ответы хранятся отдельно в `data/questions.py`.

## Животные
Описание животных и тексты программы опеки:
- `data/animals.py`

## Стикеры
ID Telegram-стикеров:
- `data/stickers.py`

---

# 🛡 Безопасность и конфиденциальность

Бот:
- не собирает чувствительные данные пользователей;
- использует `.env` для хранения токена;
- не хранит пароли и персональные данные;
- сохраняет только необходимые данные для отзывов.

Файл `.env` исключён из Git через `.gitignore`.

---

# 📈 Масштабируемость

Архитектура проекта разделена на независимые модули, что позволяет:
- легко добавлять новые вопросы;
- расширять функциональность;
- заменять SQLite на PostgreSQL при необходимости;
- поддерживать рост количества пользователей.

# 🚀 Запуск проекта

## 1. Клонирование репозитория

```bash
git clone <repo_url>
cd ZooBot_Like_Zoob_Bot
```

## 2. Создание виртуального окружения

```bash
python -m venv .venv
```

## 3. Активация окружения

### Windows

```bash
.venv\Scripts\activate
```

### Linux / macOS

```bash
source .venv/bin/activate
```

## 4. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 5. Создание `.env`

```env
BOT_TOKEN=your_bot_token
```

## 6. Запуск бота

```bash
python main.py
```
